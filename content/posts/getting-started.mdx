# Getting Started with Flikary CMS

이것은 Flikary CMS의 시작 가이드입니다.

## 주요 기능

- MDX 파일 기반 콘텐츠 관리
- REST API를 통한 콘텐츠 제공
- 서버리스 아키텍처

## 사용법

1. `content/posts/` 디렉토리에 MDX 파일 생성
2. Front matter에 메타데이터 작성
3. API를 통해 콘텐츠 조회

## API 엔드포인트

- `GET /api/posts`: 모든 포스트 목록
- `GET /api/posts/[slug]`: 특정 포스트 상세 정보
