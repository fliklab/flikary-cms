# Advanced Features

Flikary CMS의 고급 기능들을 소개합니다.

## MDX 지원

이 CMS는 MDX(Markdown + JSX)를 완전히 지원합니다.

### 코드 블록

```javascript
function hello() {
  console.log("Hello, World!");
}
```

### 수학 공식

수학 공식도 지원합니다:

$$
E = mc^2
$$

## 태그 시스템

각 포스트는 여러 태그를 가질 수 있습니다:

- `guide`: 가이드 문서
- `api`: API 관련 문서
- `tutorial`: 튜토리얼
- `reference`: 참조 문서

## 날짜 형식

날짜는 ISO 8601 형식을 지원합니다:

- `2024-07-20`: 기본 날짜
- `2024-07-20T10:30:00Z`: 시간 포함
